LKIF核心
LKIF核心法律本体论是一个与法律领域相关的本体库。它包括15个模块，每一个模块都描述了一组紧密相关的概念，从法律和常识领域。
抽象概念
最抽象的概念是在五个紧密相关的模块中定义的：顶部、地点、种族、时间和时空。
顶部LKIF顶部本体主要是基于顶级的LRI核心，但本体论的承诺较少，因为它对顶级类别的子类施加较少的限制。
地点模块部分实现相对位置理论（唐纳利，2005）在OWL DL中。
MeeLo理学，MeeLoad模块定义了诸如部分和整体的生理学概念，以及典型的环境学关系，如部分、成分、包容、成员等。
时间模块提供艾伦（1984）的时间理论的OWL DL实现。
基本概念
基本概念分布在四个模块：进程、角色、动作和表达式。
过程模块扩展了LKIF顶层本体模块，定义了变更、过程（因果变化）和物理对象。它引入了一组有限的属性来描述进程的参与者角色。
角色模块定义角色类型（认知角色、功能、人物角色、组织角色）以及角色填充与角色相关的播放属性。
动作模块描述一般用于表示动作的词汇表。动作是由某个代理（动作的参与者）执行的过程。该模块不致力于专题理论的特定理论。
表达式模块描述描述、命题和命题态度（信念、意图）、资格、陈述和媒体的词汇。它还扩展了角色模块的数量或认知角色，是规范的基础。
法律概念
这些基本集群由法律本体、法律角色和规范三个模块构成。
法律行动模块通过与诉讼、代理有关的若干法律概念，如公共行为、公共机构、法人、自然人等，扩展诉讼模块。
法律角色：法律角色模块用角色、法律职业等少量法律概念扩展角色模块。
范数范数是一个扩展，主要是在表达式模块中，规范被定义为资格。请参考可交付物1.1，对基础理论进行更深入的描述。它还定义了一些法律来源，例如法律文件、习惯法等，以及权利和权力的类型学，参见SARTROR（2006），鲁比诺等人。（2006）
框架模块
除了这些法律集群之外，还提供了两个模块，它们包括两个框架的基本词汇：修改和规则。
修改模块是时间模块和法律行动模块的扩展。时间模块以许多间隔和时刻扩展，描述了法律文件的效力和效力。动作模块用修改的类型进行扩展。这些概念在埃斯特雷亚项目的可交付项目3.2中更详细地描述。
